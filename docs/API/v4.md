# Documentation of v4 api

The generel structure of the hmm is: 

define emission and transition structures -> inference -> loss -> optimize. 

This pipelines is gathered under the class HMM which mimicks scipis regression classes. 

All stuctrues has a step function which computes the internal variables based on covariates. 


One should think about the emission and transition matrix to use the step function to compute the needed params at timestep t and then the hmm class handle the rest. 

The Emission class needs to have a distribution defined, so one should define a base distributin emisssion and use this class to expand the structure of the models
